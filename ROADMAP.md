# Sprout Roadmap

## Phase 1: MVP (v0.1)
- [ ] Flutter editor with syntax highlighting
- [ ] Rust compiler (parse `app`, `screen`, `state`, `button`)
- [ ] In-app preview (Wasm3 + Flutter bridge)
- [ ] Save/load projects

## Phase 2: Run & Install (v0.5)
- [ ] Export to APK (Android)
- [ ] Export to IPA (iOS, via Xcode export)
- [ ] One-tap install flow
- [ ] Basic AI Assistant (on-device LLM)

## Phase 3: Share & Grow (v1.0)
- [ ] `.sprout` file sharing
- [ ] Sprout Garden (dashboard of your apps)
- [ ] Template library
- [ ] Community gallery

## Phase 4: Beyond Mobile (v2.0)
- [ ] Sprout Web
- [ ] Sprout Wear (WatchOS, WearOS)
- [ ] Voice-to-app
